# World Data Explorer

Interactive world map showing real-time economic, demographic, 
and migration data by country.

## Live Demo
[link Vercel]

## Tech Stack
- React + Vite
- React Leaflet
- REST Countries API / World Bank API
- Tailwind CSS

## Features
- [ ] Interactive world map
- [ ] Country data sidebar
- [ ] Color layers by population / GDP
- [ ] Historical data timeline

## Run locally
npm install
npm run dev
